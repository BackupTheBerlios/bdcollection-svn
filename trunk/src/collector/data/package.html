Element and ElementFactory

An Element is the basic piece of knowledge of the database, and it can be saved as XML Element.toXML( DataWriter w ) and displayed as String Element.displayData(). To read, you must use the DataContentHandler that uses a ElementFactory to process the XML events. New Element can then be retrived by DataContentHandler.getData().
